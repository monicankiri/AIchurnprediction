# Customer Churn Prediction

## Description
This project predicts whether a telecom customer is likely to leave (“churn”) using Python, Scikit-learn, and Streamlit. 
It calculates the probability of churn and displays actionable predictions for business use.

## Features
- Predict churn based on 4 key customer features: `SeniorCitizen`, `Tenure`, `Contract`, `MonthlyCharges`
- Probability-based predictions
- Interactive Streamlit web app for demo purposes
- Model trained with Random Forest Classifier
